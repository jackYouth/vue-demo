## Project setup

```
npm install
首次运行vue需全局安装：
npm install -g @vue/cli
npm install -g @vue/cli-service-global
```

### vetur 插件进行代码美化

vscode 开发，下载 vetur 插件，编辑器设置中添加

```
// 保存自动化
"editor.formatOnSave": true,
// 使用vetur插件格式化vue代码
"vetur.format.defaultFormatter.html": "prettyhtml",
"vetur.format.options.tabSize": 4
```
